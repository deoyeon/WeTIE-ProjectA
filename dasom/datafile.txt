1. cnbc_sentiment.csv, guardian_sentiment.csv, reuters_sentiment.csv
https://www.kaggle.com/datasets/cashbowman/sentiment-labeled-headlines
- 데이터 개수 : 약 50000개
- Column별 설명
# : 인덱스
Headlines : 뉴스 기사 제목
Time : 기사가 게시된 날짜와 시간
Sentiment : 감정 레이블을 숫자로 표현한 값. 1~5범위이며 1은 매우 부정, 3은 중립, 5는 매우 긍정
*카테고리x. 제목에 포함된 키워드 기반으로 자동 분류 전처리가 필요.

2. goodnewseveryone-v1.0
https://www.ims.uni-stuttgart.de/en/research/resources/corpora/goodnewseveryone/
- 데이터 개수 : 5000개
- Column별 설명
headline : 뉴스기사제목
emotion : 제목에서 표현된 주요 감정(ex. Joy, anger, fear, sadness, disgust, surprise..)
experiencer : 감정을 느끼는 주체
cause : 감정을 유발한 사건이나 상황
target : 감정이 향하는 대상
cue : 감정을 나타내는 단어나 구절
reader_emotion : 독자가 제목을 읽고 인식한 감정
