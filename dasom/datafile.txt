1. cnbc_sentiment.csv, guardian_sentiment.csv, reuters_sentiment.csv
https://www.kaggle.com/datasets/cashbowman/sentiment-labeled-headlines
- 데이터 개수 : 약 50000개
- Column별 설명
# : 인덱스
Headlines : 뉴스 기사 제목
Time : 기사가 게시된 날짜와 시간
Sentiment : 감정 레이블을 숫자로 표현한 값. 1~5범위이며 1은 매우 부정, 3은 중립, 5는 매우 긍정
*카테고리x. 제목에 포함된 키워드 기반으로 자동 분류 전처리가 필요.

2. goodnewseveryone-v1.0
https://www.ims.uni-stuttgart.de/en/research/resources/corpora/goodnewseveryone/
- 데이터 개수 : 5000개
- Column별 설명
headline : 뉴스기사제목
emotion : 제목에서 표현된 주요 감정(ex. Joy, anger, fear, sadness, disgust, surprise..)
experiencer : 감정을 느끼는 주체
cause : 감정을 유발한 사건이나 상황
target : 감정이 향하는 대상
cue : 감정을 나타내는 단어나 구절
reader_emotion : 독자가 제목을 읽고 인식한 감정

3. NewsMTSC: Sentiment in English News Articles -> 파일은 업로드 안했어요
https://www.kaggle.com/datasets/fhamborg/news-articles-sentiment?select=test.jsonl
- 데이터 개수 : 11467개
- Column별 설명
sentence : 뉴스 기사 내의 한 문장
sentiment : positive, negative, neutral
source : 기사 출처
annotation_id : 감성 주석 id
*정제된 뉴스 문장 중심(기사 전체가 아니라 감성 표현이 명확히 드러나는 문장만 선별됨)
*기사 제목이 아니라 문장 단위로 구성됨 → 제목 분석용으로는 사전가공 필요
